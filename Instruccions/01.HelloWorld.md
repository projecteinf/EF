# Creació aplicació HelloWorld (Consola)

## Creació aplicació
```bash
mkdir 01.HelloWorld
cd 01.HelloWorld/
dotnet new console
```

## GIT - Ignorar fitxers binaris
```bash
echo "
01.HelloWorld/bin
01.HelloWorld/obj/
01.HelloWorld/obj/Debug/
01.HelloWorld/obj/Debug/net9.0/ " >> .gitignore
```

# Execució

```bash
pwd # ./01.HelloWorld - Ubicació del fitxer program.cs
dotnet run Program.cs
```
